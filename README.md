### 1. Clone the Repository

```bash

```

### 2. Install Dependencies


### 3. Set up Environment Variables

Create a `.env.local` file in the root directory and add your Supabase credentials:


### 4. Start the Development Server

```bash
npm run dev
```
